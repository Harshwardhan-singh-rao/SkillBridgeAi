# **App Name**: SkillBridge AI

## Core Features:

- Profile Creation & Goal Setting: Users can create profiles detailing their skills, educational background, and career aspirations to tailor the platform's AI assistance.
- AI-Powered Skill Analysis: The platform's AI analyzes user profiles to identify skill gaps and recommend personalized learning paths. This is implemented as a tool that can reason when to incorporate skills related to the user.
- Personalized Learning Path Generation: Based on the skill analysis, the AI generates a step-by-step learning path, recommending courses, projects, and resources to bridge skill gaps. This is implemented as a tool that can reason when to incorporate particular topics.
- Mentor Connection: Facilitates connections between students and industry mentors for guidance and support.
- Progress Tracking: Enables users to track their progress and visualize skill development.
- AI Career Coach: Offers an interactive AI career coach for answering student queries like 'What skills do I need for a Data Analyst role?'
- Job Matching: Recommends suitable job openings that align with the user's skills and progress.

## Style Guidelines:

- Primary color: Vibrant blue (#29ABE2) to reflect innovation and reliability in bridging college and career paths.
- Background color: Light blue (#E5F5FF), providing a clean and modern canvas that emphasizes the content's credibility.
- Accent color: Purple (#9C27B0), injecting a burst of energy and highlighting AI elements.
- Body font: 'PT Sans', a humanist sans-serif offering a modern yet friendly appeal, ensuring comfortable readability and approachability.
- Headline font: 'Space Grotesk', a contemporary sans-serif exuding a computerized, techy, scientific aura, setting a futuristic, skill-oriented tone for the platform. If longer text is anticipated, use 'Inter' for body
- Use neon-style icons, in a flat design with thin, glowing outlines to reflect the animated glow theme, especially around AI elements. These icons will be consistent and scalable.
- Implement a dynamic, single-page layout with distinct sections, utilizing curved edges and glass cards for a sleek appearance. Maintain consistency with fixed footers for vital links and CTAs.
- Employ subtle animations like parallax scrolling, typewriter effects, and floating icons to elevate the site's user experience and interactivity.
